OOP

Advanatges
- Clear structure to programs
- Code easier to maintain, reuse and debug
- helps keep code DRY (Dont Repeat Yourself)
- Allows build reusable applications with less code


Class: Class defines what an object should look like - Car
object: created based on class - Car brand (Volvo)

Object inherit all values from class

Class is blueprint for creating objects
 - use class keyword
    class MyClass:
    x = 5
- cretae object
    p1 = MyClass()
    print(p1.x)
- delete objects
    del p1
- each object from class is independant and has own copy of class properties



__init__() method
- always called when creating classes
- assigns values to object properties

e.g.
class Person:
  def __init__(self, name, age):
    self.name = name
    self.age = age

p1 = Person("Emil", 36)

print(p1.name)
print(p1.age) 

you can set default values for parameters
- init can have as many parameters as needed
